# Kiana's Smol Slime

Stacked Smol Slime, with charging dock.
30+ hours of battery life.

## Components

- ProMicro nRF52840
- LSM6DSV Module
- 401430 3.7V 150mAh Battery
- POGOPin Charger
- 30mm Tracker Straps
- User Button
